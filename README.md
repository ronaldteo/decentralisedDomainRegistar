### Install All Dependencies

npm install react-router-dom ethers@5.7 lucide-react @radix-ui/react-slot<br>
npm install -D tailwindcss@3 postcss autoprefixer tailwindcss-animate class-variance-authority clsx tailwind-merge<br>
npx tailwindcss init -p

### Launch Development server

Run `npm start` to launch the development server.
